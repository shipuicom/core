Checkboxes

Variants

Checkbox variants can be set through the class ex class="simple" takes simple , outlined , flat , and raised

Colors

Checkbox colors can be set through the class ex class="primary" takes primary , accent , warn , error , and success

Indeterminate

The indeterminate state can be set through the class ex class="indeterminate"

Disabled

The checkbox can be set to disabled through the attribute ex [disabled]
