Select

options

Array of options to display. Can be array of objects or strings.

label

Key or property to use as the display label for each option.

value

Key or property to use as the value for each option.

selectMultiple

Enable multiple selection mode.

disabled

Disable the select input.

readonly

Make the select input readonly.

inlineSearch

Enable inline search/filtering of options.

isClearable

Show or hide the clear button.

optionTemplate

Custom template for rendering options.

placeholderTemplate

Custom template for rendering placeholder if you pass a optionTemplate it will be used as a fallback.

Important to note are that because we are using inputs as the communicator we're limited to strings as values. So if the id are fx a number you need to convert it to a string.
