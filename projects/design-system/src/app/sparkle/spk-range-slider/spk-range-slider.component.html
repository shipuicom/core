Range Slider

ngModel / formControl

Two-way bindable value for the slider. Use [(ngModel)] or [formControl] for reactive forms.

min

Minimum value of the slider. [min]="0" (default: 0 )

max

Maximum value of the slider. [max]="100" (default: 100 )

step

Step size for the slider. [step]="1" (default: 1 )

disabled

Disable the slider. [disabled]="true"

color

Color theme for the slider. [color]="'primary' | 'accent' | 'warn' | 'success' | 'error'"

always-show

Always show the value indicator. [always-show]="true"

sharp

Use sharp corners for the slider. [sharp]="true"

unit

Unit of the slider. [unit]="'%'"

variation

Variation of the slider. [variation]="'base' | 'thick' | 'outlined' | 'flat' | 'raised'"
