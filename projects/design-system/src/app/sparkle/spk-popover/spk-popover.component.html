Popover

Utility component to create popovers with auto positioning using css anchor and js as a fallback.

onHover

Opens the popover on hover instead of click.

asMultiLayer

Enables multi-layer (nested) popover support.

disableOpenByClick

Disables opening the popover by clicking the trigger.

options

Popover options object: width, height, closeOnButton and closeOnEsc .
