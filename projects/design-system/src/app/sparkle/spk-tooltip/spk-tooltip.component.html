Tooltip

Content Types

Tooltips can display different types of content using the [spkTooltip] directive:

  • String : Simple text content
  • Template : Rich HTML content using ng-template
  • null/undefined : Disables the tooltip

Positioning

Tooltips automatically position themselves based on available space. They can appear:

  • Above : When there's not enough space below
  • Below : Default position
  • Left/Right : Automatically adjusted to stay within viewport

Theming

Tooltips inherit the theme from their trigger element. Apply theme classes like class="primary" , class="accent" , class="warn" , class="error" and class="success" to the trigger element.

Trigger Elements

The [spkTooltip] directive can be applied to any element. Common triggers include:

  • Buttons : For action explanations
  • Icons : For quick help text
  • Text : For additional context
  • Form elements : For validation messages

Accessibility

Tooltips automatically handle accessibility features including proper ARIA attributes and keyboard navigation. They appear on hover and disappear on mouse leave.
