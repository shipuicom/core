Dialog

Utility component to create dialogs for modals, forms, and more.

options

Dialog options object: width, height, maxWidth, maxHeight, closeOnButton, closeOnEsc, closeOnOutsideClick, class .

Data Passing

You can pass data to the dialog content component and handle the close event with a callback.

Header/Content/Footer

Use header , content , and footer attributes for layout slots. Sticky header/footer supported.
