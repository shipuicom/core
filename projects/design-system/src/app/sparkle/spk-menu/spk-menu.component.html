Menu

asMultiLayer

Enable multi-layer (nested) menu support.

searchable

Show a search input to filter menu options it's a wildcard search.

closeOnClick

Close the menu when an option is clicked (default is true), good when you want to keep the menu open for selecting multiple options.

disabled

Disable the menu trigger and menu.

Suffix

You can add a suffix attribute to an element inside the option to show a hotkey, icon, etc.
