Icons

Sizes

Icons can be set to different sizes using the class class="small" or class="large" or class="inherit"

Colors

Chip colors can be set through the class ex class="primary" takes primary , accent , warn , error and success

Color can also be overridden with css color variables both like this below or in the scss file. {{ `\cloud-warning\` }}

Icon packs

Currently we only support phospher icons, but we are working on adding more.

Search Phosphor Icons

But we do support follow icon variation types:

  • Regular (default)
  • Thin
  • Light
  • Fill
  • Bold

So the way you access the icon we use ligatures like this:

<spk-icon>cloud-warning</spk-icon>

<spk-icon>cloud-warning-thin</spk-icon>

<spk-icon>cloud-warning-light</spk-icon>

<spk-icon>cloud-warning-fill</spk-icon>

<spk-icon>cloud-warning-bold</spk-icon>
