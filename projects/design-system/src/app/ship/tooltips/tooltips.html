Tooltip

Content Types

The [shTooltip] directive supports multiple content types:

  • String : Simple text content.
  • Template : Rich HTML using ng-template .
  • null / undefined : Disables the tooltip.

Positioning

Tooltips automatically position themselves (top, bottom, left, right) based on viewport space.

Theming

Tooltips inherit their color from the trigger element. Use the color attribute on the trigger: primary , accent , warn , error , or success .

Target Elements

Apply [shTooltip] to any interactive element like buttons , icons , or form fields .
