Color picker

IMPORTANT - not production ready This is an early version of the color picker. It is not yet ready for production use. We will probably be changing the api quite a bit and could also introduce a component per render type for color selection.

Rendering Type

The visual layout of the color picker can be controlled using the renderingType input. Valid options are: hsl , grid , hue , rgb , saturation , and alpha .

Format

For the sh-color-picker-input component, the string format of the selected color can be controlled using the format input. Valid options are: rgb , rgba , hex , hex8 , hsl , and hsla .

Direction

The direction input controls the layout axis for linear renderers like hue , alpha , and saturation grids. Valid options are: horizontal and vertical .

Eye Dropper Only in natively supporting browsers

The sh-color-picker-input component includes an eye dropper button to pick a color from the screen natively. It can be toggled using the showEyeDropper input.
