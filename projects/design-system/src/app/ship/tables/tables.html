Data & Loading

Provide data via [data] and toggle a loading state using the loading attribute.

Sorting

Enable sorting by applying shSort to header cells. Track the active column with sortByColumn (two-way bindable).

Resizing

Apply the shResize directive to header cells to allow column width adjustment. Available inputs: resizable , minWidth , and maxWidth .

Sticky Columns

Use the shStickyColumns attribute ( start or end ) or apply .sticky / .sticky-end classes to header cells to pin columns.

Column Sizing

Use the size attribute on <th> elements to set initial or fixed widths (e.g., size="1fr" , size="200px" ).
