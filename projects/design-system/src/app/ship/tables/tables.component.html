Table

data

The data array to display in the table. Is a one-way binding. [data] .

loading

Boolean. Shows a loading indicator when true. [loading]="true"

sortByColumn

The column to sort by. Two-way bindable via can also be used with a model signal. [(sortByColumn)] or [sortByColumn] / (sortByColumnChange) .

shResize

Makes a column resizable. Inputs: resizable (default: true), minWidth (default: 50), maxWidth (default: null).

Can be disabled by adding class .no-resize to the table.

shSort

Makes a column sortable. Input: shSort (column name).

shStickyColumns

Makes columns sticky. Input: shStickyColumns ('start' or 'end', default: 'start').

CSS Classes

.sticky : Makes a column sticky to the left.
.sticky-end : Makes a column sticky to the right.

Size attribute

On your <th> you can add a size attribute to set the column width this is consumed by sh-table and used to calculate the grid-template-columns while also accounting for shResize.
<th size="1fr"> or <th size="min-content"> etc.
