Checkboxes

Variants

Checkbox variants can be set using the variant attribute. Valid options are: simple , outlined , flat , and raised .

Colors

Checkbox colors can be set using the color attribute. Valid options are: primary , accent , warn , error , and success .

Indeterminate

The checkbox can be set to an indeterminate state by adding the indeterminate class.

Readonly

The checkbox can be set to readonly through the readonly input (adds the readonly class).

Disabled

The checkbox can be disabled using the standard disabled attribute or [disabled] binding.
