Select

options

Array of options to display. Can be array of objects or strings. [options]="options()"

label

Key or property to use as the display label for each option passed as a string using dot notation. label="some.label.on.option.object"

value

Key or property to use as the value for each option passed as a string using dot notation. value="some.value.on.option.object"

isClearable

optionTemplate

Custom template for rendering options create a ng-template and pass it to the select component. {{ '\

placeholderTemplate

Custom template for rendering placeholder if you pass a optionTemplate it will be used as a fallback. {{ '\
