Select

Options

Provide the data source using the options attribute. This can be an array of strings or objects.

Labels & Values

If your options are objects, specify which properties to use for display and selection:

  • label : Property path to the display text (e.g., "user.name" ).
  • value : Property path to the selection value (e.g., "user.id" ).

Selection Modes

Toggle selection behavior:

  • selectMultiple : Enables picking multiple options.
  • isClearable : Adds a clear button to remove the selection (default: true).

Interaction

  • inlineSearch : Adds a search field within the dropdown to filter options.
  • readonly : Prevents selection changes while maintaining focusability.
  • disabled : Standard attribute to disable the component.

Templates

Customize rendering using ng-template :

  • optionTemplate : Custom rendering for items in the dropdown.
  • placeholderTemplate : Custom rendering for the selected state/placeholder.

Important to note are that because we are using inputs as the communicator we're limited to strings as values. So if the id are fx a number you need to convert it to a string.
