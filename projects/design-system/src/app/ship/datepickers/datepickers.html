Datepicker

Selection

Selected values are available via date and endDate . Use [(date)] and [(endDate)] for two-way binding.

Configuration

Customize the picker behavior:

  • asRange : Enable range selection mode.
  • monthsToShow : Set the number of months displayed (default: 1).
  • startOfWeek : Specify the first day of the week (0-6).
  • weekdayLabels : Provide custom labels for week days.

Input Display

Customize how dates appear in inputs:

  • masking : Set the date format mask (e.g., 'mediumDate' ).
  • size : Use the small attribute/class for a compact input.

Interaction

  • closed : Event emitted when the picker is dismissed.
  • disabled : Standard attribute to disable interaction.
