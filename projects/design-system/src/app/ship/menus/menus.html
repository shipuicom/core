Menu

Searchable

Enable a wildcard search input to filter menu options using the searchable attribute.

Behavior

Customize menu interaction:

  • asMultiLayer : Enables nested menu support.
  • closeOnClick : Determines if the menu closes when an option is selected (default: true).
  • disabled : Disables the menu trigger and content.

Slots

Add a suffix attribute to an element inside a menu option to display hotkeys, icons, or secondary information at the trailing end.
