Customization

The ShipSpotlight component is hosted within a dialog and accepts custom search items. Support for custom placeholder text, fuzzy matched categories, and custom filtering for remote API integrations is included.

Global Setup & Shortcuts

The ShipSpotlightService provides a way to integrate the spotlight seamlessly into your application. By default, a global window event listener is registered for the Cmd + K (macOS) or Ctrl + K (Windows/Linux) shortcut keys to automatically open the spotlight search overlay.

You can customize this behavior using the enableGlobalEventListener config option when registering the service:

  • enableGlobalEventListener (default: true): Controls whether the spotlight service registers the global Cmd + K / Ctrl + K keyboard listener to open the spotlight dialog. Set this to false if you prefer to manually control how/when the spotlight dialog is opened, or to avoid attaching global window event listeners.

Global Listener vs. Item Shortcuts:
The global listener (controlled by enableGlobalEventListener) listens on the window object to open the spotlight dialog modal. In contrast, individual item shortcuts (the shortcut key inside a ShipSpotlightItem) only operate while the spotlight dialog is already open, allowing users to trigger a specific option from the list (e.g., press alt+t to select a theme option).

Registering Items

Items can be registered globally or contextually using registerItems() from the ShipSpotlightService. Registered items are aggregated and displayed automatically when the spotlight opens. The method returns a cleanup function to easily unregister items when a context or component is destroyed.

Shortcuts: Each ShipSpotlightItem can be configured with an optional shortcut (e.g., "meta+e").
Warning: Do not use reserved browser hotkeys (such as Cmd+N, Cmd+T, Cmd+W). Browsers actively block the prevention of these actions, meaning the browser's default behavior will execute regardless of your application configuration. The component will throw an error if you attempt to register a reserved hotkey.

Handling Selections

Listen to the globalItemSelected signal on the ShipSpotlightService to handle actions when an item is selected from the globally-opened spotlight instance.
