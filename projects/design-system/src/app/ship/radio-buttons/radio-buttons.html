Radios

Variants

Radio variants can be set using the variant attribute. Valid options are: simple , outlined , flat , and raised .

Colors

Radio colors can be set using the color attribute. Valid options are: primary , accent , warn , error , and success .

Readonly

The radio can be set to readonly through the readonly input (adds the readonly class).

Disabled

The radio can be disabled using the standard disabled attribute or [disabled] binding.
