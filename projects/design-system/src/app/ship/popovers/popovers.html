Popover

Utility component to create popovers with auto positioning using css anchor and js as a fallback.

Hover Trigger

Use the onHover attribute or binding to open the popover on hover instead of click.

Nesting

Use the asMultiLayer attribute to enable nested popover support.

Customization

The options attribute accepts a configuration object with: width , height , closeOnButton , and closeOnEsc .
