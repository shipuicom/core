Form Field

label

Optional label for the form field. Can include icons or other elements.

prefix/suffix

Project content before or after the input using prefix or suffix slots.

placeholder

Placeholder text for the input or textarea.

hint

Optional hint text shown below the input.

error

Optional error text shown below the input when in error state.

disabled

Disables the input or textarea.

size/class

Use small , autosize , center classes for different layouts.
