Form Field

Label

Optional label for the form field. Use a <label> element inside <sh-form-field> .

Prefix / Suffix

Project content before or after the input using the prefix or suffix attributes.

Box Prefix / Suffix

Project content before or after the input but inside the input box using the boxPrefix or boxSuffix attributes on a container.

Hint

Optional hint text shown below the input using the hint attribute.

Error

Optional error text shown below the input when in an error state. Use the error attribute.

Variants

Use the variant attribute for different layouts, such as: autosize .

Sizes

Use the size attribute for different sizes or alignments, such as: small , large , or center .

Readonly

The form field can be set to readonly through the readonly input (adds the readonly class).
