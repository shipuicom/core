Sidenav

Variants

Sidenav variants can be set through the class ex default does not allow for toggling are a fairly stupid variant , simple is for when you wanna be able to toggle the size of the sidenav say you have icon state and a icons with text and overlay is for when you wanna be able to toggle the sidenav and have it overlay the content also supports dragging

disableDrag

You can pass in a boolean to disable the drag functionality for the overlay variant. [disableDrag]="true"
