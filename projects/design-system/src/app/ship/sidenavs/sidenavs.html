Sidenav

Variants

Sidenavs support multiple behaviors via the variant attribute:

  • default : Fixed position, non-toggleable.
  • simple : Support for toggling width (e.g., icon-only vs. full expanded view).
  • overlay : Overlays content when opened and supports swipe-to-close gestures.

Drag Interaction

Use the disableDrag attribute/binding to disable swipe gestures on the overlay variant.
