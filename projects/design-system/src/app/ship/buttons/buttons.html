Buttons

Variants

Button variants can be set using the variant attribute. Valid options are: simple , outlined , flat , and raised .

Sizes

Button sizes can be set using the size attribute. For example: size="small" .

Colors

Button colors can be set using the color attribute. Valid options are: primary , accent , warn , error , and success .

Loading

The button can be set to loading state by adding the loading class.

Readonly

The button can be set to readonly through the readonly input (adds the readonly class) or the standard readonly attribute.

Rotate icon

Icons within the button can be rotated by adding the rotated-icon class.

Disabled

The button can be disabled using the standard disabled attribute or [disabled] binding.
