Buttons

Variants

Button variants can be set through the class ex class="simple" takes simple , outlined , flat and raised

Sizes

Button sizes can be set through the class ex class="small"

Colors

Chip colors can be set through the class ex class="primary" takes primary , accent , warn , error and success

Loading

The button can be set to loading trough the class ex class="loading"

Readonly

The button can be set to readonly trough the attribute ex [attr.readonly] or readonly

Rotate icon

The button icons can be rotated trough the class ex class="rotated-icon"

Disabled

The button can be set to disabled trough the attribute ex [disabled]
