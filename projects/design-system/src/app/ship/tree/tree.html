Name Type Default Description

[getIcon] (item) => string | null () => null Function returning a custom icon name for a node.

Projected Elements

You can configure default folder and file icons by projecting <sh-icon> elements with specific attributes:

Element Attribute Description
<sh-icon openIcon> openIcon Icon name to display for open folder/dir nodes (e.g. folder-open).
<sh-icon closedIcon> closedIcon Icon name to display for closed folder/dir nodes (e.g. folder).
<sh-icon itemIcon> itemIcon Icon name to display for file/item nodes (e.g. file-text).
