File Upload

Files

Selected files are available via the files attribute. Use [(files)] for two-way binding.

Selection

Customize selection behavior:

  • multiple : Allows selecting more than one file.
  • accept : Restricts file types (e.g., accept=".png,.jpg" ).

Text

Customize displayed labels:

  • placeholder : Text shown when empty.
  • overlayText : Text shown during drag-and-drop.
