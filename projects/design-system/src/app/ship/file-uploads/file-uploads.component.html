File Upload

files (two-way binding)

The array of selected files. Use [(files)] for two-way binding.

multiple

Allows selection of multiple files. Use [multiple]="true" to enable.

accept

Restricts selectable file types just like the native input. Example: accept=".png,.jpg,.pdf"

placeholder

Placeholder text shown when no files are selected. Use [placeholder] to customize.

overlayText

Text shown when dragging files over the drop area. Use [overlayText] to customize.
