WYSIWYG Editor

Usage

ShipEditor is a premium, config-driven WYSIWYG rich text editor. It allows typing in design view, editing raw source code, and can serialize to html, markdown, or json configuration dynamically.

Features

  • Three output formats: HTML strings, Markdown text, and custom structured block/mark JSON.
  • ControlValueAccessor integration for Reactive and Template-driven forms.
  • Monospace code-view for editing source code directly.
  • Dynamic toolbar selection tracking.
  • Word and character count reporting in the status footer.

Block/Mark JSON Config Schema

When utilizing [format]="'json'", the value is stored as a structured array of block nodes, containing inline node spans and marks (formatting styles). 


export interface ShipEditorMark {
  type: 'bold' | 'italic' | 'underline' | 'strike' | 'link' | 'code';
  attrs?: { href?: string; target?: string };
}

export interface ShipEditorInlineNode {
  type: 'text';
  text: string;
  marks?: ShipEditorMark[];
}

export interface ShipEditorBlock {
  type: 'paragraph' | 'heading' | 'bullet-list' | 'ordered-list' | 'list-item' | 'code-block' | 'quote' | 'image' | 'hr';
  attrs?: {
    level?: number;
    src?: string;
    alt?: string;
    language?: string;
    align?: 'left' | 'center' | 'right';
  };
  content?: ShipEditorInlineNode[] | ShipEditorBlock[];
}

export type ShipEditorDocument = ShipEditorBlock[];
