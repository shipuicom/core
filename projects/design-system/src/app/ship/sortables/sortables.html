Enable Sorting

Apply the shSortable directive to any container element to enable drag-and-drop reordering.

Event Handling

Listen to the sortDrop event to update your data source when an item is moved. For tree structures, listen to the treeDrop event.

Sortable Manager (Recommended)

Vastly simplify drag-and-drop state by passing a manager to the shSortable input. Use createSortableManager() for simple and cross-list configurations, or createTreeSortableManager() for hierarchical trees.

Tree View Drag & Drop

Enable VSCode-explorer-style tree drag-and-drop by setting sortingMode="tree" and binding [treeItems] to the visible items list. Target folders must include the sortable-folder="true" attribute to accept drop-inside actions.

Drag Handles

Add the sort-handle attribute to a specific element within a sortable item to limit dragging to that specific region.
