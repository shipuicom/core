Sortable

Enable Sorting

Apply the shSortable directive to a sh-list or any container element to enable drag-and-drop reordering.

Event Handling

Listen to the sortDrop event to update your data source when an item is moved. For complex implementations, skip events and use the SortableManager (see below).

Sortable Manager (Recommended)

Vastly simplify drag-and-drop state by passing a unified manager to the shSortable input. Pass a dictionary of signals to createSortableManager(), and bind it using [shSortable]="manager" sortableGroup="listId". For single lists, simply use [shSortable]="manager". You can even provide an onBeforeDrop hook to await an API request (Promise or RxJS Observable) before accepting the drop automatically!

Handles

Add the sort-handle attribute to a specific element within a sortable item to limit dragging to that specific area.
