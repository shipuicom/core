Chips

Variants

Chip variants can be set using the variant attribute. Valid options are: simple , outlined , flat , and raised .

Sizes

Chip sizes can be set using the size attribute. For example: size="small" .

Colors

Chip colors can be set using the color attribute. Valid options are: primary , accent , warn , error , and success .

Dynamic Coloring

You can set a custom color using the CSS variable --chip-c together with the dynamic input (as an attribute or binding). This is ideal for tagging purposes where you need a wide range of colors.

Icons

Chips can contain icons using the sh-icon component. Use the suffix attribute for trailing icons.

Readonly

The chip can be set to readonly through the readonly input (adds the readonly class).

Disabled

The chip can be disabled using the standard disabled attribute or [disabled] binding.
