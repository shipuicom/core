Chips

Variants

Chip variants can be set through the class ex class="simple" takes simple , outlined , flat and raised

Sizes

Chip sizes can be set through the class ex class="small"

Colors

Chip colors can be set through the class ex class="primary" takes primary , accent , warn , error and success

Dynamic coloring

You can set a color using the css variable --chip-c together with the class dynamic . This way you can support a wide range of colors for fx tagging purposes.

Icons

Chips can contain icons using the sh-icon component. Use suffix attribute for trailing icons.

Disabled

The chip can be set to disabled through the attribute ex [disabled]
