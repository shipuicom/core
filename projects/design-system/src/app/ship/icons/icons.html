Icons

Sizes

Icons can be set to different sizes using the size attribute. Valid options are: small , large , or inherit .

Colors

Icon colors can be set using the color attribute. Valid options are: primary , accent , warn , error , and success .

Color can also be overridden with CSS color variables: {{ `\cloud-warning\` }}

We support multiple weight variations using ligatures:

  • cloud-warning (Regular - default)
  • cloud-warning-thin
  • cloud-warning-light
  • cloud-warning-fill
  • cloud-warning-bold

Icon CLI

Use our CLI tool to generate optimized icon fonts containing only the icons you use.

Generate once: ship-fg --src='./src' --out='./src/assets' --rootPath='./'

Watch mode: ship-fg --src='./src' --out='./src/assets' --rootPath='./' --watch
